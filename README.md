## 使用する時

使用するときはUserファイルやPassファイルを使いたい名前に応変更してくださいね

## コマンドについて

### コマンド一覧


/dl...保存されているログを読み込み  
/user...ユーザーの一覧を表示  
/out...チャットを停止  
/channel...チャンネルの切替  
/book...チャンネルの一覧  

コマンドの一覧は/bookからも見ることができます
